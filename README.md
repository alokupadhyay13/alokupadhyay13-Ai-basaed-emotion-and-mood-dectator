# alokupadhyay13-Ai-basaed-emotion-and-mood-dectator
This project uses a webcam to read facial expressions in real time and identify emotions like happiness, sadness, anger, fear, and surprise. By combining OpenCV with a lightweight deep-learning model, it estimates stress levels and gives instant feedback to help users track their mood and well-being. #for datasets The data sets is too big to upload so here is the link for datasets if u want https://www.kaggle.com/datasets/msambare/fer2013
